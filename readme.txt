= Create personalized experiences for your HubSpot subscribers. =

The Logic Hop HubSpot Add-on is a WordPress plugin that extends the functionality of Logic Hop. With the plugin you can display HubSpot data to your subscribers and personalize your website.

HubSpot data can be used with any Logic Hop condition while Logic Hop goals let you add or remove subscriber tags and update custom field data.

Features:

* Display HubSpot data
* Display content based on HubSpot data
* Works with any Logic Hop Condition

Requirements:

* Logic Hop 3+ Plugin
* HubSpot Account
